Add color to posts, tags, and categories

== Description ==
Add color to posts, tags, and categories. You'll see new a new metadata form on tag, category and post pages that lets you set a background and foreground color. The plugin only saves the data and doesn't change any colors on the theme.

Banner art by [Rhea Gupte](http://rheagupte.com/).
